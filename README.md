# Attendance-sheet

Its a attendance taking app, i build it for my college , basically it automates the thing regarding attendance. </br>
Class Representative is the only authorised person to use this app, that is provided with the credentials , and can login to this app.
Along with automation , it avoid the unneccessary wastage of paper, #ecofriendly.

<ul>
  <li> Course-wise Attendance </li>
  <li> Daily Attendance Report </li>
  <li> Notify Bunks Activities </li>
  <li> Share Attendace to Teacher as Excel file </li>
  <li> Provide a nice looking UI, which make it Fun :) </li>
</ul>
